Frequently Asked Questions

How do I join the server?

Simply add our server IP "Medieval1.aternos.me" to your Minecraft multiplayer list and connect!

What version of Minecraft do you support?

We support Minecraft Java Edition 1.20.1 and above. We recommend using the latest version for the best experience.

Are there any age restrictions?

Our server is family-friendly and welcomes players of all ages. We maintain a safe environment for everyone.

How do I claim land?

Use the /claim command with a golden shovel to protect your builds. Type /help claims for more information.
